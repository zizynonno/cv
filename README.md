## 概要

- 専門はバックエンドエンジニアで、現在はRuby on Railsを主戦場としています。

- 独学にて様々なサービスを開発中で、将来的には0→1で開発したサービスをマーケティングを駆使し、グロースさせていくエンジニアになりたいです。そのため、現在特にプロダクトマネージャー方面でキャリアを考えております。

- 新規開発フェーズ、改修フェーズ、保守運用フェーズを一通り経験しております。

- 学生時代にマーケティング、経営、簿記、統計、物流、プログラミングを専攻。学外ではその知見を生かすために居酒屋の外部販売や中国輸入貿易を行い、居酒屋の外部販売では1人で月商70万円。中国貿易では月収15万円を達成しました。

- ブロガーとして活動していた時期があり、ブログ上で頭のおかしいことをしたり文章で笑いをとることをしておりました。そのため、テキストコミュニケーションが増えるリモートワークにて、温かく角が立たない文章を書きつつ、短時間で正確かつ簡潔に考えを言語化し、かつちょっぴりクスっとする文章を書く能力に長けています。
